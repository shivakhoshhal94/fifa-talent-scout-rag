# Data

This project uses the **FIFA 23 Complete Player Dataset** by **Stefano Leone** on Kaggle.

The raw CSV files are intentionally **not included** in this Git repository because they are large dataset files.

## Source

- Dataset: FIFA 23 Complete Player Dataset
- Author: Stefano Leone
- Platform: Kaggle
- Original scraping source: sofifa.com

Please refer to the Kaggle dataset page and the original data source for license, usage terms, and attribution requirements.

## Dataset Highlights

The dataset contains FIFA player, team, and coach information across multiple FIFA game releases.

- Coverage: FIFA 15–23
- Time range: multiple updates between 2015 and 2023
- Players: 110+ attributes, including ratings, technical skills, physical attributes, personal information, club, league, nationality, value, wage, and position
- Coaches: 8 attributes
- Teams: 54 attributes
